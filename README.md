# Cyber Fortune Teller

A fun PowerShell project that generates random cyber-themed fortunes through an interactive command-line experience. This project was created to practice PowerShell scripting, user input handling, random output generation, and GitHub project documentation.

## 🎥 Project Demo
[Watch the Fortune Teller Demo on YouTube](https://youtu.be/xDsppdNQlmk)

## 🛠 Skills Demonstrated
- PowerShell scripting
- Interactive user prompts
- Randomized output generation
- Conditional logic and loops
- GitHub project organization and documentation

## ▶️ How to Run
1. Download or clone this repository
2. Open PowerShell
3. Navigate to the project folder
4. Run the script:
